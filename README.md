# MDC-AzurePolicyDisableExemptBook
本レポジトリは Microsoft Defender for Cloud で設定されているイニシアティブの無効化/適用除外を可視化するブックです。

# Book Image

- parameter でサブスクリプション毎に filter 可能
- 適用除外/無効化を filter 可能
